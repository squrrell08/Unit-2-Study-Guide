  • Gig Platform:Gig platforms are web or app-based marketplaces that connect businesses with providers of short-term, on-demand services.
  • src:The src attribute is used to specify the URL of a file.
  • script:The HTML script tag is a block of code that is used to execute a script, such as JavaScript.
  • Web Browser:A web browser is a software application for accessing information on the World Wide Web
  • HTML Comments:HTML comments are pieces of syntax used to document code
  • File Extension:File extensions are a way of identifying a particular type of file.
